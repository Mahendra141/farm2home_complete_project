Farm2Home is a **full-stack Farm-to-Home e-commerce application** that connects farmers directly with customers.  
The platform enables users to browse fresh vegetables and fruits, view dynamic discounts, manage cart & billing, while providing admins with secure tools to manage products and pricing.

This project is built with **React, Redux, Spring Boot, JWT, and MySQL**, following real-world production practices.

---

## ğŸš€ Project Overview

**Farm2Home** allows customers to:
- Browse products by category (Vegetables, Fruits)
- Search products with live suggestions
- View discounts applied on products
- Add products to cart
- See complete billing breakdown
- Proceed to delivery details

Admins can:
- Securely log in (JWT based)
- Add, edit, or disable products
- Manage base price, discounts & demand level
- View discounted prices in real time
- Control product availability
